# QB
基于现有题库（Word格式）生成题目，默认自带一个习概的题库，有以下功能：
1.解析word文档为题库
2.随机从题库中选取题目
3.将已结算的题目重新拉回题库

## ***gemini pro 2.5 编写***

## **合法Word格式：**
```
    一、单选题
        1.	社会主义民主政治的本质特征是（ ）。
        A. 人民当家作主
        B. 党的领导
        C. 依法治国
        D. 全过程人民民主
        正确答案: A
        2.	党的二十大指出，（ ）是社会主义民主政治的本质属性。
        A. 人民当家作主
        B. 党的领导
        C. 依法治国
        D. 全过程人民民主
        正确答案: D
```

***仅支持处理以下四种题型：***
单选题、多选题、填空题、判断题
